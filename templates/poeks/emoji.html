What's this?
Done

emojiboard is an alternate keyboard of symbols which are supported by the iPhone and Mac OS X, but aren't available on the standard keyboard.

How's it work?

Tap and hold on a symbol to bring up the standard pasteboard interface. To copy more than one glyph at a time, paste them into the scratchboard box, then copy its contents.

Why not just tap a key to copy?

This is not supported by WebKit, and it will probably never be, for security reasons. Bummer!

Emojiboard by Neven Mrgan

To install Emojiboard, tap the + button below and choose Add to Home Screen.

Once installed, the app will be automatically updated if there are changes. Sweet!

emojiboard is an iPhone web app. Please visit this page on an iPhone or iPod touch to install Emojiboard. Later!
